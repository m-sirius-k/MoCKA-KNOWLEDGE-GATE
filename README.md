## システムアーキテクチャ（日本語版）

MoCKA Insight System は、AI ガバナンス研究を「実行 → 検証 → 記憶 → 再利用」という循環で支える多層構造です。

- **MoCKA Core**  
  研究実行と一次成果物の生成を担う中心エンジン。

- **Knowledge Gate**  
  推論・判断・検証文脈を制度的記憶として保存するレイヤー。

- **Transparency Layer**  
  監査・証跡・検証可能性を提供する透明性レイヤー。

- **External Brain**  
  外部知識・他プロジェクトとの統合を担う知識統合レイヤー。

- **Civilization Layer**  
  ガバナンス思想・制度設計・長期的文脈を保持する文明レイヤー。

これらのレイヤーが連携することで、研究は実行されるだけでなく、説明可能・検証可能・再利用可能な形で蓄積されます。

## 研究知識フロー（日本語版）

1. **実行（Execution）**  
   MoCKA Core が研究を実行し、ログ・推論・成果物を生成する。

2. **検証（Verification）**  
   結果を検証し、前提・方法・判断の妥当性を確認する。

3. **制度化（Institutionalization）**  
   検証済みの成果物を Knowledge Gate に保存し、文脈と理由を付与する。

4. **再利用・ガバナンス（Reuse & Governance）**  
   保存された知識は次の研究や制度設計に再利用され、文明として蓄積される。
